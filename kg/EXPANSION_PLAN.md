## Node Quality Standards

Each node MUST include:

1. **Metadata**:
   - `node_id`: Consistent naming (type.lang.name_level)
   - `type`: Appropriate node type
   - `label`: Clear English label
   - `cefr_level`: A1, A2, or B1
   - `source`: CEFR Companion, PCIC, etc.

2. **Content**:
   - `diagnostics`: Form, function, usage constraints
   - `prompts`: 2-3 practice prompts per node
   - `examples`: Authentic corpus examples with citations

3. **Metadata Fields** (where applicable):
   - `frequency`: Zipf scores for Lexeme nodes
   - `frequency_source`: Corpus source
   - `corpus_examples`: PRESEEA or other corpus citations
   - `prerequisites`: Links to prerequisite nodes

4. **Strand Assignment**:
   - Meaning-Input: Comprehension-focused (listening/reading)
   - Meaning-Output: Production-focused (speaking/writing)
   - Language-Focused: Explicit study (grammar, vocabulary)
   - Fluency: Automaticity practice (mastered content)
